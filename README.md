## Algorithm Visualizer
Simple application that visualizes different sorting algorithms implemented in C++23 using wxWidgets.
### Currently supported algorithms
- Merge Sort
- Quick Sort
- Bubble Sort
### Building
```
git clone --recurse-submodules git@github.com:Casper9474/Algorithm-Visualizer.git AlgorithmVisualizer
cd AlgorithmsVisualizer
cmake --workflow --preset release
```
